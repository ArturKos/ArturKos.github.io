CHIP-8 Emulator

A faithful CHIP-8 virtual machine emulator that recreates the 1970s interpreter originally designed for the COSMAC VIP microcomputer. Built with C++17 and Allegro 5 for graphics, sound, and input handling, the emulator implements the complete CHIP-8 architecture including memory management, CPU registers, display rendering, timers, and keyboard input. It supports loading ROM files to run a variety of classic games.

Features

  • Full 35-opcode instruction set: Complete implementation of the original CHIP-8 specification
  • 4096 bytes of memory: Faithful memory layout with program space starting at address 0x200
  • 16 general-purpose registers: V0 through VF, plus index register and program counter
  • 16-level call stack: Supports deep subroutine nesting
  • 64x32 monochrome display: XOR-based sprite drawing with collision detection
  • Dual timers: Delay timer and sound timer, both operating at 60 Hz
  • 16-key hexadecimal keypad: Mapped input for all CHIP-8 key codes
  • Built-in font sprites: 4x5 pixel hex digit characters (0-F) loaded into memory
  • ROM file loading: Load and run classic CHIP-8 game ROMs

Technical Details

  • Language: C++17
  • Graphics/Sound/Input: Allegro 5
  • Build system: CMake 3.10+
  • Platform: Linux

Build Instructions

Requirements: CMake (3.10+), Allegro 5

git clone https://github.com/ArturKos/CHIP-8.git
cd CHIP-8
mkdir build && cd build
cmake ..
make
