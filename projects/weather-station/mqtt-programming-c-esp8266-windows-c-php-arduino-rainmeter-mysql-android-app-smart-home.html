Weather Station

A full-stack IoT weather monitoring system that measures temperature and humidity both indoors and outdoors. The system spans the entire stack — from ESP8266 firmware reading hardware sensors, through a Raspberry Pi server with PHP/MySQL backend, to multiple client frontends including a web dashboard, Android app, and Windows desktop widget. The system also includes a weather forecast module that predicts temperature and humidity using weighted historical averages blended with recent trend extrapolation.

Architecture

  • ESP8266 firmware — reads DHT11 (indoor) and DHT22 (outdoor) sensors every 300 seconds, sends data via HTTP POST over WiFi
  • Raspberry Pi server — hosts PHP/MySQL backend, receives and stores sensor data with API key authentication
  • Web dashboard — PHP-generated pages with CanvasJS chart visualization of historical data
  • Forecast dashboard — dark-themed responsive UI with Chart.js, showing multi-day temperature and humidity predictions based on historical data analysis
  • Android app — WebView-based client displaying real-time and historical data
  • Windows widget — Rainmeter desktop skin for at-a-glance monitoring

Features

  • Dual sensor monitoring — indoor (DHT11) and outdoor (DHT22) temperature and humidity
  • Automatic data collection every 5 minutes with WiFi retry logic (15 attempts)
  • RESTful API with API key authentication for secure data ingestion
  • Historical chart visualization using CanvasJS on the web dashboard
  • JSON data export for real-time readings
  • Weather forecast — up to 7-day predictions using 70% weighted historical average (same date ±3 days across years) + 30% recent 3-day trend extrapolation
  • Multi-platform clients — web browser, Android, and Windows desktop
  • Grid-powered with sensors connected via twisted pair cable (~2m)

Hardware Components

  • Raspberry Pi 4 (4 GiB RAM) — server hosting PHP/MySQL
  • ESP8266 — WiFi microcontroller reading sensors and transmitting data
  • DHT11 — indoor temperature and humidity sensor
  • DHT22 — outdoor temperature and humidity sensor (higher precision)

Technical Details

  • Firmware: C/C++ (Arduino framework for ESP8266), serial monitoring at 115200 baud
  • Backend: PHP 7.x with MySQL database, RESTful endpoints
  • Frontend: PHP-rendered HTML with CanvasJS charts, responsive design
  • Android: Java WebView-based client loading server-rendered pages
  • Forecast: PHP JSON API with statistical prediction engine, Chart.js frontend with hourly detail tables and trend visualization
  • Widget: Rainmeter skin for Windows 10 desktop integration
  • Database schema: Per-station tables with columns for indoor/outdoor temperature and humidity
