English Learning Slideshow

Screenshot

A self-hosted English vocabulary tool running on a local Apache/PHP server. The app displays a full-screen slideshow of English learning images — grammar charts, vocabulary cards, and phrase guides — advancing automatically every 60 seconds. A Word of the Day widget picks a random entry from a 575-word Polish–English dictionary on every page load or button press.

The interface is built as a single-page application with a dark, animated background featuring a drifting grid and ambient glow effects. Images are served dynamically by a PHP backend that scans the image directory and returns the list as JSON.

Features

  • Auto-advance slideshow: Images rotate every 60 seconds with an animated progress bar
  • Word of the Day: Random word drawn from a 575-entry Polish–English dictionary
  • Shuffle: Randomise image order at any time
  • Fullscreen mode: One-click distraction-free viewing
  • Fade transitions: Smooth crossfade with background preloading
  • Keyboard navigation: ← / → to browse, Space to pause, F for fullscreen
  • Touch swipe: Swipe left/right on mobile devices
  • Dark animated UI: Drifting grid background, ambient glow blobs, Inter + JetBrains Mono fonts

Technical Details

  • Frontend: HTML5, CSS3 (custom dark theme, CSS animations), Vanilla JavaScript
  • Backend: PHP — scans image directory and returns file list as JSON
  • Fonts: Inter, JetBrains Mono (Google Fonts)
  • Images: 156 English learning JPEG photos
  • Dictionary: 575 Polish–English word/phrase pairs in plain text
  • Server: Apache + PHP on Raspberry Pi (local network)

Source code: github.com/ArturKos/english-learning
