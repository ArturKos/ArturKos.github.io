BigNumber Library

An arbitrary-precision integer arithmetic library implemented in C++, where numbers are stored as doubly-linked lists of 8-bit nodes. This design allows integers of virtually unlimited size, far exceeding the 64-bit boundary of native types. The library provides full operator overloading for natural arithmetic syntax, bit-level operations, number-theoretic functions, and multiple output formats. All functionality is validated by 43 unit tests with zero memory leaks confirmed via Valgrind.

Features

  • Full operator overloading: +, -, *, /, %, ==, !=, <, <=, >, >=
  • Bit operations: Shift left, shift right, bitwise NOT
  • Primality testing: is_prime() with 6k+-1 optimization algorithm
  • Integer square root: Computed via binary search
  • Multiple input types: Construct from uint8_t, uint16_t, uint32_t, uint64_t
  • Output formats: Binary, decimal, and hexadecimal display
  • 43 unit tests: Comprehensive test suite using the GoogleTest framework
  • Zero memory leaks: Verified with Valgrind
  • Doxygen documentation: Full API documentation generated from source

Technical Details

  • Language: C++11 or later
  • Data structure: Doubly-linked list of 8-bit nodes
  • Testing: GoogleTest framework
  • Build system: CMake 3.14+
  • Documentation: Doxygen

Build Instructions

Requirements: CMake (3.14+), GoogleTest

git clone https://github.com/ArturKos/BigNumber.git
cd BigNumber
mkdir build && cd build
cmake ..
make
./tests    # run unit tests
