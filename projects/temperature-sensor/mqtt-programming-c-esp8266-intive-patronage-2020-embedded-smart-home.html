Temperature Sensor

Intive Patronage 2020

An ESP8266-based temperature sensor library supporting both analog and digital sensors, developed as part of the Intive Patronage 2020 program (Szczecin, embedded track). The library uses MQTT for real-time data transmission to a central server, enabling efficient temperature monitoring within a smart home ecosystem.

Features

  • Dual sensor support — both analog and digital temperature sensors
  • MQTT integration for real-time publish/subscribe data transmission to a central server
  • Configurable sampling intervals for adjustable monitoring frequency
  • WiFi connectivity with automatic reconnection handling on the ESP8266
  • Smart home ecosystem — works alongside RGB LED controllers and other IoT devices
  • Calibration support for accurate readings across sensor types

Technical Details

  • Language: C (Arduino/ESP8266 framework)
  • Microcontroller: ESP8266
  • Protocol: MQTT for publish/subscribe messaging
  • Sensors: Analog (ADC-based) and digital temperature probes
  • Part of: Intive Patronage 2020 smart home ecosystem (Szczecin)
