Arkanoid Game

After AI Refactoring

Legacy Version

Screenshots

A polished C++ implementation of the classic Arkanoid arcade game, built with Allegro 5 for graphics, audio, and input handling. The player controls a paddle to deflect a ball into a 10x5 grid of destructible tiles, each with an HP system requiring up to 3 hits to destroy. The game wraps the gameplay in a full retro experience: an animated CRT-styled intro screen, a name input prompt, level progression with freshly randomized grids, three distinct power-ups, particle effects, and a persistent top-10 high score table.

Features

  • Retro CRT intro screen: falling pixel-art title bricks, rainbow shimmer, starfield background, bouncing ball, decorative bottom bricks
  • Name input screen with blinking cursor and CRT scanline overlay
  • Destructible tile grid: 10x5 layout with per-tile HP system (up to 3 HP), beveled rendering with highlight/shadow edges and visible crack lines as tiles take damage
  • Level progression: clearing all tiles advances to the next level with a freshly randomized grid
  • 3 power-ups with 25% drop chance: wider paddle, slow ball, and extra life — each with a 600-frame duration timer and on-screen progress bar
  • Particle explosion system: per-tile particles with gravity simulation, color variation, and fade effects
  • Ball motion trail: fading tail rendered behind the ball for visual polish
  • Screen shake triggered on tile destruction for impact
  • Paddle angle control: ball angle changes based on where it strikes the paddle (left third, center, right third)
  • Persistent high scores: top-10 leaderboard saved to scores.dat in binary format with animated row fade-in
  • Resizable window with aspect-ratio-preserving scaling and fullscreen toggle (F)
  • Cross-platform: Runs on both Linux and Windows

Technical Details

  • Language: C++
  • Graphics/Audio/Input: Allegro 5
  • Build system: CMake 3.2+
  • Platforms: Linux, Windows

Build Instructions

Requirements: CMake (3.2+), Allegro 5

git clone https://github.com/ArturKos/arkanoid.git
cd arkanoid
mkdir build && cd build
cmake ..
make
