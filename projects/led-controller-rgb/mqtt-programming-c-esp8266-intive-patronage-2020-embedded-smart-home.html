LED Controller RGB

Intive Patronage 2020

An ESP8266-based RGB LED strip controller developed as part of the Intive Patronage 2020 program (Szczecin, embedded track). The library provides intuitive color control via HSV-to-RGB conversion and uses MQTT for real-time data transmission to a central smart home server.

Features

  • HSV-to-RGB color conversion for intuitive color selection and smooth transitions
  • MQTT integration for real-time data transmission to a central server
  • Brightness management with programmable intensity levels
  • Lighting effects — create custom color sequences and patterns
  • Smart home ecosystem — works alongside temperature sensors and other IoT devices
  • Configurable WiFi with automatic reconnection handling

Technical Details

  • Language: C (Arduino/ESP8266 framework)
  • Microcontroller: ESP8266
  • Protocol: MQTT for publish/subscribe messaging
  • Color model: HSV input converted to RGB PWM output
  • Part of: Intive Patronage 2020 smart home ecosystem (Szczecin)
