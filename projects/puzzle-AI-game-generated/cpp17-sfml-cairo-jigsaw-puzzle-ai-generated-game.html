Jigsaw Puzzle Game

Fully AI-Generated

Screenshots

A desktop jigsaw puzzle game built entirely with AI assistance using C++17, SFML 2.5 for graphics and input, and Cairo for rendering realistic interlocking puzzle pieces via Bezier curves. The game features three escalating difficulty levels — from 12 pieces (Easy) to 192 pieces (Nightmare) — with smooth drag-and-drop gameplay, snap-to-grid alignment, and audio feedback.

Features

  • Classic jigsaw pieces with interlocking tabs and blanks rendered via Cairo Bezier curves
  • 3 difficulty levels: Easy (12 pieces), Medium, and Nightmare (192 pieces)
  • Drag-and-drop with visual feedback — snap-to-grid and drop shadows
  • 5 animated screens: intro, name/difficulty selection, gameplay, victory, and high scores
  • Persistent score tracking with difficulty-based filtering
  • Toggleable sound effects
  • All images and sounds embedded at compile time — fully self-contained binary
  • Custom image support — add PNG/JPG/BMP files and rebuild

Technical Details

  • Language: C++17
  • Graphics: SFML 2.5, Cairo 1.16+
  • Build system: CMake 3.16+
  • Resource embedding: Python code generator converts assets to C++ byte arrays
  • Architecture: Modular codebase — rendering, puzzle generation, scoring, and UI components

Build Instructions

Requirements: CMake (3.16+), pkg-config, SFML (2.5+), Cairo (1.16+)

git clone https://github.com/ArturKos/puzzle-AI-game-generated.git
cd puzzle-AI-game-generated
mkdir build && cd build
cmake ..
make
