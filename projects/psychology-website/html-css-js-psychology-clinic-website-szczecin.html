Psychology Clinic Website

Screenshots

A professional website built for a psychological testing and consultation practice (Pracownia Badan Psychologicznych) based in Szczecin, Poland. The clinic, operating since 2009, offers psychological assessments for drivers, machine operators, firearms licensing, occupational health evaluations, court expert testimony, and psychological support services.

The website serves as the clinic's primary digital presence, providing potential clients with detailed service descriptions, office location with Google Maps integration, contact information, and a photo gallery of the facilities.

Features

  • Responsive design: Fully mobile-friendly layout adapting to all screen sizes
  • Service showcase: Detailed pages for each type of psychological assessment offered
  • Google Maps integration: Embedded map showing the clinic's location in central Szczecin
  • Schema.org structured data: MedicalBusiness markup for improved search engine visibility
  • Photo gallery: Office facility images showcasing the professional environment
  • Contact section: Phone number, office hours, and location details
  • Google Analytics: Visitor tracking and traffic analysis
  • SEO optimized: Proper meta tags, structured data, and semantic HTML

Technical Details

  • Languages: HTML, CSS, JavaScript
  • Hosting: GitHub Pages
  • Analytics: Google Analytics
  • SEO: Schema.org MedicalBusiness structured data
  • Maps: Google Maps embed
