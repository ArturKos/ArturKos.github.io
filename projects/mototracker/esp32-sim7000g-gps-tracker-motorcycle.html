MotoTracker — Motorcycle GPS Tracker

Screenshots

A complete motorcycle GPS tracking system built around the LilyGO T-SIM7000G — a compact board combining an ESP32 microcontroller with a SIM7000G cellular/GPS modem, an SD card slot, and an 18650 battery holder. The tracker sends GPS coordinates, speed, temperature, and battery voltage to a self-hosted server over 2G/NB-IoT/CAT-M cellular.

The MotoTracker web dashboard runs on a Raspberry Pi (Apache + PHP + MySQL) and displays the day's route on an interactive Leaflet map, color-coded by speed. Rides can be replayed in real time or exported as GPX files for use in Strava, Garmin Connect, or other apps.

Firmware Features

  • Live mode: GPS always on, sends 1 point/second via HTTP GET with keep-alive TCP connection
  • Batch mode: deep sleep 60 s between fixes, RTC memory batches 5 points per wake — low power
  • SD offline buffer: up to 100 000 points (~27 h at 1 pt/s) stored on SD when network is unavailable; auto-recovered on reconnect
  • Battery protection: auto-sleeps below 3.3 V to prevent over-discharge
  • Compact HTTP protocol: short parameter names (k, v, la, lo, s, t, h, b, ts) to minimise cellular overhead

Dashboard Features

  • Leaflet map: route drawn as a polyline colour-coded green→red by speed
  • Ride stats: total distance, duration, average and max speed, temperature, battery voltage
  • Ride replay: animated playback with adjustable speed (1×–20×)
  • Ride history: list of the last 50 rides with date, point count, and duration
  • GPX export: download any day's ride as a standard GPX track file
  • Live tracking: auto-refreshes every 2 s when viewing today's date

Technical Details

  • Hardware: LilyGO T-SIM7000G (ESP32 + SIM7000G, GPS 1 Hz, SD card, 18650 battery)
  • Firmware: Arduino C++ — TinyGSM, ArduinoHttpClient
  • Connectivity: 2G GSM / NB-IoT / CAT-M (HTTP, no TLS — avoids 2–5 s TLS overhead on SIM7000G)
  • Backend: PHP 7.4+, MySQL 5.7+, Leaflet.js 1.9.4, Font Awesome 6.5
  • Server: Raspberry Pi running Apache on local network with DDNS
  • 3D case: custom-designed enclosure for the LilyGO T-SIM7000G with power button — PLA or PETG
