Puzzle Game

Engineering Degree Project

A jigsaw puzzle game developed as an engineering degree project, built with C++ Builder 6.0 and the Windows VCL framework. The game features multiple screens, persistent save/load functionality, MIDI audio playback, player profiles, and a leaderboard system.

Features

  • Multiple game screens — animated AVI intro, game loading, main gameplay, results, and high scores
  • Save and load game progress with custom binary serialization
  • MIDI audio playback via Windows MCI (Media Control Interface)
  • Custom data structures — StosPuzli stack-based system for puzzle piece management with linked nodes
  • Player profiles and persistent leaderboard tracking
  • Configurable difficulty with varying piece counts
  • JPEG image support for puzzle source images
  • GDI rendering with VCL Canvas for smooth graphics

Technical Details

  • Language: C++ (Borland C++ Builder 6.0)
  • Framework: Windows VCL (Visual Component Library)
  • Graphics: GDI via VCL Canvas, JPEG decoding
  • Audio: MIDI playback through Windows MCI interface
  • Persistence: Custom binary format for saved games, separate player data files
  • Architecture: Multi-form application with 7 forms (intro, loading, gameplay, results, scores, name entry, about)
